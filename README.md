# gitbook-plugin-drainage
微信公众号引流插件, 在文章末尾添加微信公众号图片

* 插件配置
```javascript
{
    "plugins": [
        "drainage@git+https://github.com/xnzone/gitbook-plugin-drainage.git#0.0.1"
    ],
    "pluginsConfig": {
        "drainage" : {
            "image": "https://gitee.com/ixnzone/img-bed/raw/master/wechat.jpg",
            "height": "411",
            "width": "540"
        }
    }
}
```
